# Spark Heroku App

Peticiones HTTP usando Spark Framework y React para despliegue de informacion

### Demo

Ingresar a: https://sparkherokuapp.herokuapp.com/
### Prerequisitos
Se recomienda contar con las siguientes versiones instaladas:
```
version de java: 15
version de compilador: 1.8
Apache maven: 3.63 
```
